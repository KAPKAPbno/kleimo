# KleimoRobot - бот для добавления ватермарок

Телеграм-бот для наложения ватермарок на фото и видео.

## Установка
1. Установите FFmpeg (необходим для работы с видео).
2. Склонируйте репозиторий.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Создайте переменную окружения `BOT_TOKEN`.
5. Запустите: `python main.py`.

## Команды
- `/text` — изменить текст
- `/color` — изменить цвет
- `/size` — изменить размер
- `/mode` — режим (одиночный/плитка)
- `/pos` — позиция
